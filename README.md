# 🍽️ Chez Nany — Site Web (Officiel, pas pour l'instant)

Site vitrine développé pour le restaurant **Chez Nany**.

## 🌐 Aperçu

Ce projet a pour objectif de fournir une présence en ligne moderne et élégante au restaurant, tout en offrant une navigation fluide et une expérience utilisateur optimisée.

## ✨ Fonctionnalités

- 🏠 Page d'accueil immersive
- 📖 Présentation du restaurant
- 📄 Menu accessible en ligne
- 📍 Page "Où nous trouver" avec Google Maps
- 🕒 Affichage dynamique des horaires (ouvert / fermé en temps réel)
- 📱 Design responsive (mobile & desktop)
- 📅 Système de réservation (en cours de développement)

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚧 État du projet

Le site est actuellement en phase de finalisation. Certaines fonctionnalités, notamment le système de réservation, sont encore en cours de développement.
